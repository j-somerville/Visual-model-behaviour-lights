# Visual-model-behaviour-lights
R scripts and code for visual modelling of fish and stats analysis for behavioural responses to light.
Each fish behaviour folder contains r scripts and csv files for analysing behaviour responses to light (skate and plaice - time spent stationary, catsharks - light interaction behaviour).
The quantum-catch folder includes r scripts and csv files on how to calculate visual stimulation values towards different light colours for catsharks, plaice and skate.
